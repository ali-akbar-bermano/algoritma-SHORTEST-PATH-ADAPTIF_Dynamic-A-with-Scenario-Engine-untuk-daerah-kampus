    Tentang Aplikasi
    Navigasi kampus UNIB menggunakan algoritma Dynamic A* dengan:

    🏫 29 Node lokasi kampus
    🛣 48 Edge jalur jalan
    🗂 4+ Skenario kondisi kampus
    ⚙ Bobot edge dinamis

    Legenda:
    🟢 Normal · 🟡 Sibuk (×2) · 🟠 Berlubang (×1.6) · 🔴 Ditutup (∞)

    Cara Pakai:
    1. Pilih Awal & Tujuan → Cari Rute
    2. Tekan 📊 Bandingkan untuk lihat semua skenario
    3. Aktifkan Edit Bobot Jalan → klik jalan
    4. Gunakan ✏️ Gambar Jalur untuk ukur panjang
    5. Klik baris di tab Jalan → zoom ke jalan itu
    6. Tab 🎭 Skenario untuk kelola & tambah skenario
    Sumber Data Peta
    📍 Tile Map: OpenStreetMap — peta terbuka dunia, dirender via Leaflet.js
    📐 Koordinat Node: Koordinat GPS asli 29 lokasi kampus UNIB, dicatat manual menggunakan Google Maps
    🛤 Geometri Jalan: Polyline koordinat diambil dari dataset OSM & diverifikasi ulang secara manual untuk akurasi sempurna
    🧭 Routing: Jalan baru menggunakan OSRM API untuk mendapatkan geometri yang mengikuti jalur jalan nyata
    📏 Jarak: Dihitung menggunakan formula Haversine dari koordinat GPS, bukan garis lurus

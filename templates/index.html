    Tentang Aplikasi
    Aplikasi navigasi kampus UNIB menggunakan algoritma Dynamic A* dengan:

    🏫 29 Node lokasi kampus
    🛣 48 Edge jalur jalan
    🗂 4 Skenario kondisi kampus
    ⚙ Bobot edge dinamis (kondisi jalan, permukaan, skenario)

    Legenda Kondisi Jalan:
    🟢 Normal · 🟡 Sibuk (×2) · 🟠 Berlubang (×1.6) · 🔴 Ditutup (∞)

    Cara Pakai:
    1. Pilih Awal & Tujuan di tab Rute
    2. Klik "Cari Rute" → rute muncul di peta
    3. Aktifkan Edit Mode (kanan atas peta)
    4. Klik jalan → ubah kondisi
    5. Cari rute ulang → algoritma mempertimbangkan kondisi baru
